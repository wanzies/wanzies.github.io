Open the deployed URL in **Chrome** 2. Tap the **⋮ menu** → **Add to Home screen** 3. Tap **Install** --- ## 🤖 AI Recipe Suggestions (Claude API) FridgeSync uses the [Anthropic Claude API](https://anthropic.com) to suggest recipes based on your current fridge contents, prioritising items expiring soonest. ### Setup 1. Get a free API key at [console.anthropic.com](https://console.anthropic.com) 2. Open FridgeSync → tap **Recipes** tab 3. Enter your key in the **Claude API Key** field and tap **Save** 4. Your key is stored in `localStorage` on your device only — never sent anywhere except Anthropic's API > The app uses `claude-sonnet-4-5` with a structured JSON system prompt. Missing ingredients can be added to your shopping list with one tap. --- ## 📷 Barcode Scanning Barcode scanning uses the [BarcodeDetector Web API](https://developer.mozilla.org/en-US/docs/Web/API/BarcodeDetector) where available (Chrome on Android, some desktop browsers), with a manual-entry fallback for iOS Safari. Product information is fetched from the [Open Food Facts API](https://world.openfoodfacts.org/api) — free, no key required. | Platform | Barcode Method | |---|---| | Chrome (Android / desktop) | `BarcodeDetector` API (native, fast) | | iOS Safari | Manual barcode entry fallback | | Firefox | Manual barcode entry fallback | --- ## 🔔 Notifications FridgeSync supports two notification channels: **Browser Push (Web Push API)** - Enable in Settings → Notifications - Requires HTTPS (GitHub Pages / Netlify) - Works on Android Chrome and desktop browsers - iOS Safari supports Web Push from iOS 16.4+ when installed as a PWA **WhatsApp Fallback** - No account or API key required - Generates a pre-formatted `wa.me` link - User taps the link to send the message to their household WhatsApp group - Works on any device with WhatsApp installed --- ## 👨‍👩‍👧 Multi-User & Household Sharing ``` 1. First member creates a household → gets a 6-digit code (e.g. 482910) 2. Share the code with family members 3. Others open the app → Join Household → enter the code 4. All members see the same fridge in real time ``` > **Note on data sync:** The current version stores data in `localStorage` on each device. For real-time cross-device sync, connect a Firebase Firestore backend — the data model and security rules are documented in [`FridgeSync_TechSpec.docx`](docs/FridgeSync_TechSpec.docx). The app UI and logic are fully ready for Firestore; only the data layer needs wiring up. --- ## 🔧 Firebase Integration (optional — for real-time sync) To enable live cross-device sync, replace the `localStorage` calls in `index.html` with Firebase Firestore: ```javascript // 1. Add Firebase SDK to // // // 2. Initialise with your Firebase config const app = firebase.initializeApp({ /* your config */ }); const db = firebase.firestore(); // 3. Replace saveState() with Firestore writes // db.collection('households').doc(state.householdId).set(state) // 4. Replace loadState() with onSnapshot listener // db.collection('households').doc(hid).onSnapshot(snap => { state = snap.data(); render(); }) ``` Full Firestore data model, security rules, and index configuration are in the [Technical Specification](docs/FridgeSync_TechSpec.docx). --- ## 🗂️ Planning Documents The full product documentation lives in the `/docs` folder: | Document | Description | |---|---| | [`FridgeSync_PRD.docx`](docs/FridgeSync_PRD.docx) | Product Requirements — personas, feature scope, success metrics | | [`FridgeSync_UserStories.docx`](docs/FridgeSync_UserStories.docx) | User Stories & Acceptance Criteria (12 stories, ready for Jira/Notion) | | [`FridgeSync_TechSpec.docx`](docs/FridgeSync_TechSpec.docx) | Technical Specification — architecture, APIs, Firebase setup, testing checklist | --- ## 🛣️ Roadmap - [ ] Firebase Firestore real-time sync - [ ] Firebase Cloud Messaging push (server-side fan-out) - [ ] Multiple fridge support (fridge + freezer) - [ ] Item photo upload - [ ] Nutritional info from Open Food Facts - [ ] Dark/light theme toggle - [ ] Export inventory as CSV --- ## 🙏 Credits & APIs Used | Service | Purpose | Cost | |---|---|---| | [Open Food Facts](https://world.openfoodfacts.org) | Barcode product lookup | Free, no key | | [Anthropic Claude](https://anthropic.com) | AI recipe suggestions | Pay-per-use | | [Google Fonts](https://fonts.google.com) — Syne + DM Sans | Typography | Free | | Web Push API | Browser push notifications | Free (browser-native) | | BarcodeDetector API | Camera barcode scanning | Free (browser-native) | --- ## 📄 License MIT — do whatever you like with it.
